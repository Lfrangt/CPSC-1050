## 4. Key Concept Summary (Ch 5)

### CPU Components
- **CPU:** Control Unit (CU) + Arithmetic Logic Unit (ALU) + Registers  
- **Instruction Register (IR):** holds the current instruction being executed  
- **Program Counter (PC):** holds the **address** of the next instruction (not the instruction or data)

### Memory
| Memory Type | Volatile? | Random Access? |
|-------------|-----------|----------------|
| RAM         | Yes       | Yes            |
| ROM         | No        | Yes            |
| Disk        | No        | Yes            |

### Bus
- Group of wires carrying: **address**, **data**, **control** signals.

### Parallelism
- **Pipelining:** instruction execution in stages (efficiency).  
- **Task-level parallelism:** different processors, different tasks, at the same time.
